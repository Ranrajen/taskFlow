# TaskFlow - React To-Do & Notes App

A responsive, component-based productivity dashboard built with React.js.

## Features
- Add, complete, and delete tasks.
- Create and delete personal notes.
- Light / Dark mode toggle.
- Built using React Context API for global state management.

## How to run locally
1. Clone the repo: `git clone https://github.com/Ranrajen/taskFlow.git`
2. Navigate to folder: `cd taskFlow`
3. Install dependencies: `npm install`
4. Start the app: `npm start`
